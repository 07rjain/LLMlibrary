    Class Conversation

    Stateful conversation wrapper that handles history, tool execution, persistence, and running token/cost totals.

    Example

    const conversation = await client.conversation({
      sessionId: 'support-1',
      system: 'Be concise.',
    });

    await conversation.send('Summarise the issue.');
